What is a Command Line / CLI App?

Applications with text based interface, invoked from a terminal

Example: ls rails npm etc.

Why write a CLI app?

To Automate tasks like:

  • Reading, writing creating files
  • Generating project templates
  • Deploying applications

Why NodeJS?

  • You can use any language!
  • Easier / prettier than writing Bash scripts
  • Rich ASCII libraries commander.js chalk progress
  • because this is a JavaScript Meetup!
